# ChristocentricTraderBot

A Telegram bot that posts Good Morning messages, verses, trading ideas, and more.

## Setup
1. Clone this repo
2. Add your `BOT_TOKEN` and `CHAT_ID` to `.env`
3. Install dependencies:
pip install -r requirements.txt
   `
4. Run the bot:
   `
   python bot.py
   `

Files
- content.json → Library of messages
- log.json → Rotation tracker
- bot.py → Main bot script
- .env → Secrets
- requirements.txt → Dependencies
